Being There Website Redesign

client:
Breda University of Applied Sciences research department

The Being There Project is a collaborative project between Breda University and University of Monterrey which researches the effect of VR to generate empathy for Latin American migrant women journeying through Mexico to the United States.

This is their current website.

Client Brief

Create a prototype for a new visually appealing and interactive website to replace the current version. Eventually this will be built into a new website.


  • The website is meant to generate funds for a migrant centre. It should have a donate function and funnel the users towards this.
  • The website should generate empathy and engage the audience with the women who are experiencing this.

  • The website should also showcase the current VR project being developed for the migrant centre.
